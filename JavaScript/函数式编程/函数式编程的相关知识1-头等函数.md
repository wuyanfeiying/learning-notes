### First-class Function (函数是一等公民/头等函数）
> 当一门编程语言的函数可以被当作变量一样用时，则称这门语言拥有`头等函数`。
- 有如下特性:
  - 函数可以被赋值给一个变量;
  - 函数可以被当作参数传递给其他函数;
  - 函数可以作为另一个函数的返回值。

#### 把函数赋值给变量
> 即使你的函数有自己的函数名称，你仍然可以用这个变量名去调用它。
```js
  // 将匿名函数赋值给变量
  const fn = function() {
    console.log("fn_bar");
  }
  fn(); // fn_bar

  // 将有名称的函数赋值给变量
  const fn2 = function name() {
    console.log("fn2_bar");
  }
  fn2(); // fn2_bar
```
